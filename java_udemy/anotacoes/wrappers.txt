### WRAPPERS ###
- São os tipos orientados a objetos dos tipos primitivos.
- Possuem funções específicas e pre-definidas para eles.

### Os Wrapers ###

- Byte = byte
- Short = short
- Integer = int
- Long = long

---Booleanos---

Boolean = boolean

---Fracionados---

- Double = double
- Float = float

---------- Por que usa-los? -------------
Eles habilitam funções específicas para manipulação de dados que necessitam do tipo deles.
ex

Integer pode fazer com que eu possa transformar um número em String com o métodos toString, habilitado a partir dele.
