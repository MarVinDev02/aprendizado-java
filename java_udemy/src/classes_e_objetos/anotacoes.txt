### Classe ###

- Uma classe define um bloco de código.
- Orientação a objetos tenta trazer programação para um mundo real.
- Atributos e métodos são MEMBROS da classe.

### Membro da Classe ###

- Como acessar os membros de uma classe?
ex:
objeto.membro
       \ ponto é o acessor do membro da classe.

### Métodos ###

- São comportamentos de uma classe.
- Estão interligados aos algoritmos.
- São os comportamentos que uma classe irá ter.
